# Assignment 6: Basic Enterprise Connectivity

## Objective
To configure two routers connected via WAN and verify connectivity using RIPv2.

## Topology
- 2 Routers
- 2 Switches
- 4 PCs
- 1 DHCP Server on LAN 1

## IP Addressing
LAN 1: 192.168.1.0/24
LAN 2: 192.168.2.0/24
WAN: 10.0.0.0/30

Router1:
- 192.168.1.1
- 10.0.0.1

Router2:
- 192.168.2.1
- 10.0.0.2

## Routing Protocol
RIPv2

## Verification
Successful ping between LAN1 and LAN2 PCs.
